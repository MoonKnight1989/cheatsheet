HTML Table Reference

HTML Basics

Tag Name Description
<!DOCTYPE html> HTML Declaration This tells the text editor what type of code to expect in the document. Text editors such as VS Code will automatically add this with the HTML boilerplate automation
<head> Head Tag This is where all of the metadata, cookies and scripts go. It is not visible on the page and contains information to help search engines (and humans) understand the page better
<body> Body Tag This is where all of the visible information is added. Anything in the body tag will be visible to the user on the page. It is organised using other tags such as <div> and <section> to help with readability and to target CSS selectors at specific elements
<title> Title Tag This tag is contained in the <head> section and contains the title of the page. THis is used by Google to display in the SERP for users to see what thre page or article is all about.
